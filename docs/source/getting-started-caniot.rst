Getting Started
=======================================

Setup
-----

.. note:: for the first use or after Factory Reset,Caniot Application need access to the Internet to Download and  ``Registering`` your software.If you do not have access to the Internet, the application will be closed after 10 seconds.

    1. To begin, download and install Caniot Application for your OS from the `Caniot download page`_

    2. With the CaniotBox plugged into your computer, start the Caniot Application. If this is the first time you are using your caniot Tool or after a Factory Reset,
    the Caniot Application will ask you to enter your ``application Key`` . we recommend you to read the license first.

    3. after you have entered the application key and the registration was successful, you have to select a work Directory, in this Directory named ``Caniot_data``  you will found later all Settings and Configurations stored during run time from Caniot Application.
